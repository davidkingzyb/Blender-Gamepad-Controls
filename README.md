# Blender Gamepad Controls 游戏手柄控制插件

**Blender gamepad controller for sculpting and texture paint workflow.**

2026 by DKZ

fork from [OhMyKing/Blender-Gamepad-Controls](https://github.com/OhMyKing/Blender-Gamepad-Controls) 重新映射按键功能以适应雕刻和纹理绘制工作流.  
内置 [zeth/inputs](https://github.com/zeth/inputs/tree/master) 包,直接下载zip包开箱即用.

- F5 enable 开启(切换工作模式需重新开启)  

- 左摇杆：Rotate 旋转视角
- 右摇杆：Move 平移视角
- 十字键: View 视图切换
- L1/L2: Scale 缩放
- L3: Front View 正视图
- R1: Sample 拾色器(在Texture Paint中使用,windows系统api,暂不支持其他系统)
- R2: Erase 橡皮/Direction 雕刻方向
- X/Y: Brush size 笔刷大小
- A/B: Brush Strength 笔刷强度
- -/+: undo/redo 重做
